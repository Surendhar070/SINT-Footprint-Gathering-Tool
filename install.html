Installation Guide

Install and run the OSINT Footprint Gathering Tool on Windows using OSINT.exe. Quick start and troubleshooting.

Quick Start – Windows

Install the application

  1. Download OSINT.exe
    • Go to the Downloads page
    • Download OSINT.exe (or the setup installer ZIP)
  2. Run the application
    • If you have OSINT.exe: double‑click to run. No installation needed.
    • If you have the installer ZIP: extract it, then run OSINT.exe from the extracted folder.
    • If you use a setup wizard (e.g. OSINTTool_Setup_*.exe): run it, follow the steps, then launch from Start Menu or Desktop shortcut.
  3. Launch
    • Open OSINT.exe from your chosen folder, or use the Start Menu / Desktop shortcut if the installer created one.

Windows 10/11 (64‑bit) required. Internet connection needed for footprinting and Firebase login.

Troubleshooting

DNS/Network Issues

Problem: Analysis fails with network errors

Solution:

  • Check internet connection
  • Verify firewall settings
  • Some corporate networks may block DNS queries

MTNL (Mahanagar Telephone Nigam Limited) Users

Problem: Tool fails with DNS/timeout errors, WHOIS lookup failures, or "getaddrinfo failed" on MTNL broadband

Solution:

  • DNS: MTNL's default DNS can be slow or block some queries. Switch to public DNS:
    • Google: 8.8.8.8, 8.8.4.4
    • Cloudflare: 1.1.1.1, 1.0.0.1
    • Change in Windows: Settings → Network & Internet → Ethernet/Wi‑Fi → Edit (DNS) → Manual → add the above.
  • Firewall: MTNL modems/routers may restrict outbound DNS or certain ports. Allow OSINT.exe (or the OSINT tool) in Windows Firewall.
  • WHOIS: WHOIS over default MTNL routes can time out. Using the DNS change above often fixes it. The app also suppresses WHOIS console errors.
  • Router: If issues persist, try rebooting the MTNL modem/router and re-running the tool.
