About the Tool

Professional OSINT footprinting for security researchers, investigators, and analysts.

Entity Types

Entity Capabilities
Domain DNS records, WHOIS, subdomains, SSL certificates, hosting provider, CMS detection.
URL HTTP headers, page content, technology stack, security headers, robots.txt.
Email MX, SPF/DKIM/DMARC, domain analysis, disposable and role-based detection.
IP Address Geolocation, ASN, ISP, reverse DNS, port checks, threat intel placeholders.
Mobile Number Country code, format analysis, international variations.
Person / Username Social media discovery, GitHub, username enumeration, cross-platform identity.
Organization Domain discovery, email patterns, subsidiaries, infrastructure mapping.
IOC / Threat Intel Indicators (IP, domain, URL, hashes), malicious checks, C2 relationships.

Technology

Built with Python, Tkinter GUI, matplotlib/NetworkX for graphs, and Firebase for authentication. Supports JSON, TXT, and PDF export.
